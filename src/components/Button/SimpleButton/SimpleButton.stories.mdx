## Installation

```jsx

```

### Loading state

Set/unset the `isLoading` prop to toggle loading state

<Preview>
  <Story name="Loading state">
    <Box
      display="grid"
      gridAutoFlow="column"
      gridGap="10px"
      justifyContent="start"
    >
      {simpleButtonSizes.map(size => (
        <SimpleButton isLoading={true} size={size}>
          Button
        </SimpleButton>
      ))}
    </Box>
  </Story>
</Preview>

```jsx
<SimpleButton isLoading={true}>Button Text</SimpleButton>
```

### Loading state with custom loading text

Set the `loadingText` prop to set custom loading text
_Only works when `isLoading` prop set to true_

<Preview>
  <Story name="Loading state with custom loading text">
    <Box
      display="grid"
      gridAutoFlow="column"
      gridGap="10px"
      justifyContent="start"
    >
      {simpleButtonSizes.map(size => (
        <SimpleButton isLoading={true} loadingText="Loading" size={size}>
          Button
        </SimpleButton>
      ))}
    </Box>
  </Story>
</Preview>

```jsx
<SimpleButton isLoading={true} loadingText="Loading">
  Button Text
</SimpleButton>
```
