# TextButton

---

## Installation

```jsx
import { TextButton } from "@bit/xto10x.common/Button";
```

### Loading state

Set/unset the `isLoading` prop to toggle loading state

<Preview>
  <Story name="Loading state">
    <Box
      display="grid"
      gridAutoFlow="column"
      gridGap="10px"
      justifyContent="start"
    >
      {textButtonSizes.map(size => (
        <TextButton isLoading={true} size={size}>
          Button
        </TextButton>
      ))}
    </Box>
  </Story>
</Preview>

```jsx
<TextButton isLoading={true}>Button Text</TextButton>
```

### Loading state with custom loading text

Set the `loadingText` prop to set custom loading text
_Only works when `isLoading` prop set to true_

<Preview>
  <Story name="Loading state with custom loading text">
    <Box
      display="grid"
      gridAutoFlow="column"
      gridGap="10px"
      justifyContent="start"
    >
      {textButtonSizes.map(size => (
        <TextButton isLoading={true} loadingText="Loading" size={size}>
          Button
        </TextButton>
      ))}
    </Box>
  </Story>
</Preview>

```jsx
<TextButton isLoading={true} loadingText="Loading">
  Button Text
</TextButton>
```
